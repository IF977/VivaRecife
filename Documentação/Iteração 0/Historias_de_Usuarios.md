### VivaRecife
> User Stories

#### USUÁRIO
```
Buscar Bairros

Como um usuário
Eu quero ver a lista de bairros
Para que eu possa visualizar todos os bairros possíveis da cidade e possa escolher entre eles
```

```
Descobrindo o Bairro

Como um usuário
Eu quero ver a lista de estabelecimentos e eventos no meu bairro
Para que eu possa saber das possibilidades de eventos e locais próximos
```

```
Avaliação

Como um usuário
Eu quero pesquisar sobre determinado estabelecimento e/ou evento 
Para que eu possa avaliá-lo
```

```
Complementar Informações

Como um usuário
Eu quero entrar em contato com o site
Para que eu possar adicionar informações não existentes sobre um estabelecimento e/ou evento
```

```
Adicionar Informações

Como um usuário
Eu quero entrar em contato com o site
Para que eu possar adicionar estabelecimentos e/ou eventos não existentes a um bairro
```

```
Avaliação Colaborativa

Como um usuário
Eu quero poder avaliar um estabelecimento e/ou evento
Para que os outros usuários e visitantes saibam como foi minha experiência com ele
```

```
Conhecendo a Ideia

Como um usuário
Eu quero acessar a página “sobre” 
Para que eu possa saber sobre a origem do projeto
```

#### VISITANTE

```
Descobrindo o Site

Como um visitante
Eu quero fazer uma rápida busca no site
Para que eu possa ter uma informação relevante sobre determinado bairro
```
